How to play

Train your brain with this memory card game and catch all the pokemons! There are two identical images on the game board.
Start the game by flipping a card. Then try to find another card that has the same pokemon as the first.
If you can't find a pair, the flipped cards will be flipped back with the face down.
Try to remember those pokemons as it becomes easier to find pairs the longer you play!

Enjoy... You gotta catch 'em all!
